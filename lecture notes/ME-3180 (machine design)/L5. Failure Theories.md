# L5. Failure Theories


Static failure theories best practices:

|             | **Even**     | **Uneven**       |
| ----------- | ------------ | ---------------- |
| **Ductile** | Von Mises    | Max Shear Stress |
| **Brittle** | Coulomb Mohr | Modified Mohr    |
Use this table to select a failure theory for a given application.

An "even" material has the same material properties in compression and tension. Most ductile materials are even. 

> For quizzes and tests, **if the material is ductile, assume it is even.** **If the material is brittle, assume it is uneven.**



### Von Mises (2D)

$$\sigma ' = \sqrt{\sigma_1 ^2 + \sigma_3^2 - \sigma_1\sigma_3}$$

$$\sigma ' = \sqrt{\sigma_x^2 + \sigma_y^2 - \sigma_x\sigma_y + 3\tau_{xy}^2}$$

The safety factor here is:

$$N = \frac{S_y}{\sigma'}$$

### Max Shear Stress Theory

$$N = \frac{S_{ys}}{\tau_\max}=\frac{0.50 S_y}{\tau _\max} = \frac{S_y/2}{(\sigma_1-\sigma_3)/2}= \frac{S_y}{\sigma_1-\sigma_3}$$

This is always more conservative than von mises.

### Max Normal Stress Theory

### Working around Modified Mohr

If you don't want to use modified mohr, you can "save time" by seeing if $\sigma_3 > 0$. 

If $\sigma_3>0$, just use the [Max Normal Stress Theory](#Max%20Normal%20Stress%20Theory).

If $\sigma_3<0$, you can use [Coulomb-Mohr Theory](#Coulomb-Mohr%20Theory) for a conservative result. Not really sure how to get the less conservative result tbh. 

If this is too annoying just use the [Modified Mohr](#Modified%20Mohr) theory.
