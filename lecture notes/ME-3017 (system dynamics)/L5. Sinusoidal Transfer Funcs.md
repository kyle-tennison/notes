# L5. Sinusoidal Transfer Funcs

The transfer function $G(s)$ can be changed to be:

$$G(s) \to G(j\omega)$$

> Something about $s = \sigma + j \omega$, where $\sigma$ is the transient portion, which is zero for a steady state sine oscillation.


For a sine input:

$$u(t) = P \sin(\omega t), \quad \Rightarrow \mathscr L\{u(t)\}=U(s)$$

The gain is:

$$|G(j \omega) | = \left| \frac{Y(j \omega)}{U(j \omega)} \right|$$

and the phase shift is:

$$\angle G(j\omega) = \angle \left(\frac{Y(j\omega)}{U(j\omega)}\right) = \angle Y(j\omega) - \angle U(j\omega)$$

> General complex operations would be good to note on the exam.

Then, the *steady state output* $y(t)$ is:

$$\boxed{y(t) = |G(j\omega)|P\sin(\omega t + \angle G(j\omega))}$$

The output is always guaranteed to also be a sine function.

If $u(t)$ is not a steady state function, *the equation above is not complete*. You need to include the transient response:

$$\text{full solution = steady-state solution + transient response solution}$$

> ig we don't care about this in this class.

Even if $u(t)$ *is* only sinusoidal, $y(t)$ will (usually? always?) still have a transient portion for when the model is "spinning up" to the input. It will eventually reach its own steady state oscillation. 

### Phase and Amplitude of Imaginary Numbers

$$|a+jb| = \sqrt{a^2+b^2}$$

$$\angle (a+jb) = \tan^{-1}\left(\frac ba\right)$$

> this one is only true for $a>0$. Just draw it out to get the angle lowkey.

## 1st Order Systems

If you have a transfer of the format:

$$G(s) = \frac{1}{Ts+1} \Rightarrow \frac{1}{Tj\omega+1}$$

The gain is:

$$20\log|G(j\omega)| = -20 \log \sqrt{T^2\omega^2+1}$$

And the phase shift is:

$$\angle G(j\omega) =-\tan^{-1}\left(\omega T\right)$$

At $\omega T\approx 1$, the plot takes a noticeable turn:
![[image-124.png]]

The slope before and after is:

$$0 \text{ db/decade for } \omega T\ll 1$$
$$-20 \log(\omega T) \text{ db/decade for } \omega T\gg 1$$

This turning point is the "corner/cutoff frequency" $\overline \omega$, which is:

$$\overline \omega = \frac 1T$$
## Properties of Bode Plots
### Inverse Transfer Functions

If $H(s) = 1/G(s)$, the bode plot of $H$ will be symmetric over the horizontal axis:

![[image-125.png]]

That is:

$$20 \log|H(j\omega)| = -20 \log|G(j\omega)|$$

$$\angle H(j\omega) = -\angle G(j\omega)$$

### Decomposing Transfer Functions

If $G(s)=G_1(s)G_2(s)$, then the gain can be written as:

$$20 \log|G(j\omega)| = 20 \log|G_1(j\omega)| + 20 \log|G_2(j\omega)|$$

And

$$\angle G(j\omega) = \angle G_1(j\omega) + \angle G_2(j\omega) $$

"Multiplication in the s domain is addition in the bode plot."

## Vibration Analysis

The transmissibility is just the gain(?). Usually wanna minimize this.

If you have a system with two masses, and you know it'll be harmonic, you can put say:

$$x_1= A \sin(\omega t), \qquad x_2 = B\sin(\omega t)$$

Then, solve the system. If $A/B>0$, the masses move *with* each other. If $A/B<0$, the masses move against each other. This will vary depending on the natural frequency used in the solution.
