# L6. Linearization

If you approximate a function using the Taylor series to the 1st order:

$$f(x) \approx f(\bar x)+ \left.\frac{df}{dx}\right|_{x=\bar x}(x-\bar x)$$
This converts the nonlinear function into a linear approximation around $\bar x$.

## Properties of Linear Systems

Responses add (superposition):

$$u_1 \to y_1, \quad u_2 \to y_2 \quad \Rightarrow \quad u_1+u_2 \to y_1+y_2$$

Scaling the input scales the output (homogeneity):

$$ku \to ky$$

A sinusoidal input gives a sinusoidal output at the *same* frequency; only amplitude and phase change:

$$u(t)=P\sin(\omega t) \quad \Rightarrow \quad y(t)=|G(j\omega)|P\sin(\omega t + \angle G(j\omega))$$
