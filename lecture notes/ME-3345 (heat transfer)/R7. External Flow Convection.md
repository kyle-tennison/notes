# R7. External Flow Convection

*Forced convection* is from forced flow, like that from a fan, and does *not* include movement from buoyancy forces—that would be *natrual convection.*

The following assumes no phase change, *external flow* where boundary layers can freely develop, and relatively low speed fluid motion.

From [R6. Convection](R6.%20Convection.md), we wish to find either $Nu$ or $\overline {Nu}$ so that we can find $h$ and hence heat transfer. If we are interested in mass transfer, we wish to find $Sh$ or $\overline {Sh}$. From dimensional analysis, we know:

$$Nu_x = f(x^*, Re_x, Pr), \qquad \overline{Nu}_x = f(Re_x, Pr)$$

The $x$ subscript denotes functions of position relative to the beginning of the boundary layer $x^*=0$. Similar can be done for concentration, but I won't cover that here (covered on p. 386).

## Parallel Flow over a Flat Plate

![[image-119.png|349]]

For a *flat plate in parallel, flow* like shown above, the Nusselt Number can be determined using the following equation:

$$\overline{Nu}= C \,Re^m_L\,Pr^n$$

where $C$, $m$, and $n$ are experimentally determined coefficients that are generally independent of the fluid's nature and depend primarily on the *type* of flow and the surface geometry. 

When using this equation, we are making the assumption of *constant fluid properties*. However, because these properties vary with temperature, we evaluate this equation at the **film temperature** $T_f$ which is defined by:

$$T_f \equiv \frac{T_s+T_\infty}{2}$$
^film-temp

In the following equations, the $x$ represents the distance from the leading edge of the surface.
### Laminar Case

The flow will be laminar when $Re \lesssim 5 \times 10^5$. If that is true, then the boundary layer thickness is given by:

$$\delta = \frac{5x}{\sqrt{Re_x}}$$

The Reynold's Number in this case is:

$$Re_x = \frac{u_\infty x}{\nu}$$

The wall shear stress is:

$$\tau(x) = \mu\left. \frac{\partial u}{\partial y}\right|_{y=0}\approx 0.0.332 u_\infty \sqrt{\rho \mu u_\infty /x}$$

The *local* friction coefficient is:

$$C_{f,x} = \frac{\tau(x)}{\rho u_\infty ^2/2} \approx 0.664 Re_x^{-1/2}$$

The thermal boundary layer $\delta_t$ is given by:

$$\frac{\delta}{\delta_t} = Pr^{1/3} = \left( \frac{\nu}{\alpha} \right)^{1/3}$$

And the local Nusselt number is:

$$Nu_x = \frac{h_xx}{k}\approx0.332 Re_x^{1/2}Pr^{1/3}\qquad \text{when } Pr \gtrsim0.6$$

Taking the average of this gives the average Nusselt number:

$$\overline{Nu}_x = \frac{\bar h_xx}{k}= 0.664Re_x^{1/2}Pr^{1/3}, \qquad Pr \gtrsim0.6$$

### Turbulent Case

This applies for $5\times 10^5 \le Re_x \lesssim 10^7$. 

The local friction coefficient is:

$$C_{f,x} \approx 0.0592 Re_x^{-1/5}$$

The boundary layer thickness is:

$$\delta \approx 0.37x Re_x^{-1/5}$$

The thermal boundary layer is approximately equal to the velocity boundary layer ($\delta_t \approx \delta$) just because the flow is so turbulent and there is so much mixing.

The local Nusselt number is:

$$Nu_x \approx 0.0296 Re_x^{4/5}Pr^{1/3}$$

You could find the average of this, but because the turbulent region is generally preceded by a laminar region, we usually need to consider a mixed case.

### Mixed Case

This is the common case for when there is both laminar and turbulent flow on the same surface.

$$\overline{Nu}_L = (0.037Re_L^{4/5}-A)Pr^{1/3}$$

where $A$ is determined by the critical Reynolds number $Re_{x,c}$. 

$$A = 0.037Re_{x,c}^{4/5} - 0.664Re_{x,c}^{1/2}$$

The critical Reynolds number is generally taken to be $Re_{x,c}\approx 5\times 10^5$; unless specified otherwise, this evaluates to $A=871$. 

If the flow starts as turbulent (at $x=0$), then use $A=0$. There is no laminar region to consider.

### Uniform Surface Heat Flux

When there is a uniform heat flux off of the surface (instead of a uniform temperature), you instead use:

$$Nu_x \approx 0.453 Re_x^{1/2}Pr^{1/3}, \qquad Pr \gtrsim0.6$$
$$Nu_x \approx 0.0308Re_x^{4/5}Pr^{1/3}, \qquad 0.6 \lesssim Pr \lesssim 60$$

This is the case when:

$$q''_x = h_x(T_s(x)-T_\infty) = \rm const$$
If you are trying to find the surface temperature, you can use:

$$T_s(x) = T_\infty +\frac{q_x''}{h_x}$$

## Cylinder in Cross Flow

![[image-120.png]]

- The forward stagnation point is where $\partial u / \partial y =0$.
- If $Re < 2 \times 10^5$, separation occurs around $\theta \approx 80^\circ$ 
- If $Re > 2 \times 10^5$, the flow is turbulent and separation is delayed until $\theta \approx 130^\circ \text{-} 140^\circ$ 
- In this **case**, the Reynold's number is $Re_D = \frac{VD}{\nu}$


The Nusselt number is a function of $\theta$:

$$Nu_\theta = \frac{h(\theta)D}{k}$$


From this,

![[image-121.png]]

The empirical fit for this is:

$$\overline{Nu}_D=\frac{\bar hD}{k}= CRe_D^mPr^{1/3},\qquad Pr \gtrsim0.7$$

this is known as *Hilbert's Correlation*. You can get the heat transfer from this then with:

$$q=\bar h A_s(T_s-T_\infty)$$

where $A_s$ is the surface over which air flows over; i.e. $\pi DL$ or $2 \pi r L$.
