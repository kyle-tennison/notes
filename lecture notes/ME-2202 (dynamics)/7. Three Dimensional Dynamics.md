# 7. Three Dimensional Dynamics

## 7/3 Fixed-Axis Rotation

For a rigid body rotation about a fixed axis, a point on the body has:

$$\mathbf v = \boldsymbol \omega \times \mathbf r$$

$$\bf a = \dot {\boldsymbol \omega} \times r + \boldsymbol \omega \times (\boldsymbol \omega \times r)$$

(Same as 2D so far)

## 7/5 Rotation about a Fixed Point


![[IMG_0863.jpeg]]


Two rotations *do not* obey the standard law of vector addition, as shown in the figure above. However, infinitesimal rotations *do* obey by this principle. 

Mathematically:

$$d \boldsymbol \theta = d \boldsymbol \theta_1 + d \boldsymbol \theta_2$$

even though that

$$\boldsymbol \theta \ne \boldsymbol \theta_1 +\boldsymbol \theta_2 $$

Because of this principle, it *is* true that:

$$\frac{d \boldsymbol \theta}{dt} = \frac{d \boldsymbol \theta_1}{dt} + \frac{d \boldsymbol \theta_2}{dt}$$

or

$$\boldsymbol \omega = \boldsymbol \omega_1 + \boldsymbol \omega_2$$

This is the **addition theorem**. In terms of frames, we can say:

$$\boldsymbol\omega_{C/I}=\boldsymbol\omega_{C/B} + \boldsymbol\omega_{B/I}$$

You can prove this using the [vector derivative formula](5.%20Plane%20Kinematics%20of%20Rigid%20Bodies.md#^vector-derivative-formula). 

The angular acceleration is:

$$\begin{aligned}
\boldsymbol\alpha_{C/I} &= \dot{\boldsymbol\omega}_{C/I} \\
&=\left(\frac {d}{dt}\right)_I(\boldsymbol\omega_{C/B}+\boldsymbol\omega_{B/I})\\
&=\left(\frac {d}{dt}\right)_I(\boldsymbol\omega_{C/B}) + \boldsymbol\alpha_{B/I}
\end{aligned}
$$

$\boldsymbol\omega_{B/I}$ can be converted directly to $\boldsymbol\alpha _{B/I}$ because it is already in the $I$ frame (and the derivative is w.r.t. the $I$ frame.) For $\omega_{C/B}$, we reapply the [vector derivative formula](5.%20Plane%20Kinematics%20of%20Rigid%20Bodies.md#^vector-derivative-formula) (this time in its pure form) ...


$$\left( \frac{d\mathbf V}{dt}\right)_{A} = {\left( \frac{d\mathbf V}{dt}\right)_{B} + \boldsymbol \omega_{B/A} \times \mathbf V }$$

... to get:

$$\begin{aligned}
\left(\frac{d \boldsymbol\omega_{C/B}}{dt} \right)_{I} &= \left(\frac{d \boldsymbol\omega_{C/B}}{dt} \right)_{B} + \boldsymbol\omega_{B/I}\times \boldsymbol\omega_{C/B}
\\\\&=\boldsymbol\alpha_{C/B} + \boldsymbol\omega_{B/I} \times \boldsymbol\omega_{C/B}
\end{aligned}$$


Altogether, this gives:

$$
\begin{aligned}
\boldsymbol\alpha_{C/I} = \boldsymbol\alpha_{C/B}+ \boldsymbol\omega_{B/I}\times \boldsymbol\omega_{C/B} + \boldsymbol\alpha_{B/I}
\end{aligned}
$$

### Rigid Body Velocity

As we saw with 2D:

$$\mathbf r_{P/I} =\mathbf r_{B/I}+\mathbf r_{P/B}$$

![](excalidraw-2025-11-19-17.02.37.excalidraw.svg)
%%[🖋 Edit in Excalidraw](excalidraw-2025-11-19-17.02.37.excalidraw.md)%%



The velocity form of this is:

$$\mathbf {\dot r}_{P/I} =\mathbf {\dot r}_{B/I}+\mathbf {\dot r}_{P/B}$$

$$
\begin{aligned}
\left ( \frac{d \mathbf r_{P/I}}{dt}\right)_I &= \underbrace{\left ( \frac{d \mathbf r_{B/I}}{dt}\right)_I}_{\text{simply }\mathbf v_{B/I}} + \underbrace{\left ( \frac{d \mathbf r_{P/B}}{dt}\right)_I}_{\text{needs transform}}\\\\\
\mathbf v_{P/I}&=\mathbf v_{B/I} +\underbrace{\left ( \frac{d \mathbf r_{P/B}}{dt}\right)_B}_{\text{simply } \mathbf v_{P/B}}+ \boldsymbol\omega_{B/I}\times \mathbf r_{P/B}
\end{aligned}
$$
$$\boxed{\mathbf v_{P/I}=\mathbf v_{B/I} + \mathbf v_{P/B} + \boldsymbol\omega_{B/I}\times \mathbf r_{P/B}}$$

And, if $P$ is fixed on the body, $\mathbf v_{P/B}=0$, and the equation reduces to:

$$\mathbf v_{P/I} = \mathbf v_{B/I} + \boldsymbol\omega_{B/I}\times \mathbf r_{P/B}$$

which is identical to [Relative Velocity](5.%20Plane%20Kinematics%20of%20Rigid%20Bodies.md#Relative%20Velocity) equation for 2D bodies.

### Rigid Body Acceleration 

Considering the same body shown in the prior section, if we want to find the acceleration $\mathbf a_{P/I}$, we may apply the vector derivative formula once more to the equation we found for $\mathbf v_{P/I}$:


$$
\begin{aligned}
\left(\frac{d \mathbf v_{P/I}}{dt}\right)_I &= \left(\frac{d}{dt}\right)_I(\mathbf v_{B/I} + \mathbf v_{P/B}+\boldsymbol \omega_{B/I}\times \mathbf r_{P/B})
\\\\
\mathbf a_{P/I}&=\left(\frac{d \mathbf v_{B/I}}{dt}\right)_I + \left(\frac{d \mathbf v_{P/B}}{dt}\right)_I + \left(\frac{d}{dt}\right)_I(\boldsymbol\omega _{B/I} \times \mathbf r_{P/B})
\\\\
&= \mathbf a_{B/I} + \left[ \left(\frac{d \mathbf v_{P/B}}{dt}\right)_B + \boldsymbol\omega_{B/I}\times \mathbf v_{P/B} \right] +\left(\frac{d}{dt}\right)_I(\boldsymbol\omega _{B/I} \times \mathbf r_{P/B})
\end{aligned}
$$


The $\left(\frac{d}{dt}\right)_I(\boldsymbol\omega _{B/I} \times \mathbf r_{P/B})$ term requires some special treatment. The [product rule for cross products](https://en.wikipedia.org/wiki/Cross_product#:~:text=.-,Differentiation,-%5Bedit%5D) states:

$${\frac {d}{dt}}(\mathbf {a} \times \mathbf {b} )={\frac {d\mathbf {a} }{dt}}\times \mathbf {b} +\mathbf {a} \times {\frac {d\mathbf {b} }{dt}}$$
[^1]


Applied to this case:

$$\begin{aligned}
\left(\frac{d}{dt}\right)_I(\boldsymbol\omega _{B/I} \times \mathbf r_{P/B}) &= \left(\frac{d \boldsymbol \omega_{B/I}}{dt}\right)_I \times \mathbf r_{P/B} + \boldsymbol \omega_{B/I} \times \left(\frac{d \mathbf r_{P/B}}{dt}\right)_I\\\\
&=\boldsymbol\alpha_{B/I}\times \mathbf r_{P/B} +  \boldsymbol \omega_{B/I} \times \left[ \left(\frac{d \mathbf r_{P/B}}{dt}\right)_B+\boldsymbol \omega_{B/I} \times \mathbf r_{P/B}  \right]\\\\
&= \boldsymbol\alpha_{B/I}\times \mathbf r_{P/B} +  \boldsymbol \omega_{B/I} \times(\mathbf v_{P/B} + \boldsymbol\omega_{B/I}\times \mathbf r_{P/B})

\end{aligned}
$$

Then, when substituted into the original equation:


$$
\begin{aligned}
\mathbf a_{P/I}&= \mathbf a_{B/I} + \left[ \left(\frac{d \mathbf v_{P/B}}{dt}\right)_B + \boldsymbol\omega_{B/I}\times \mathbf v_{P/B} \right] +\left[\boldsymbol\alpha_{B/I}\times \mathbf r_{P/B} +  \boldsymbol \omega_{B/I} \times(\mathbf v_{P/B} + \boldsymbol\omega_{B/I}\times \mathbf r_{P/B}) \right] \\\\
&=\mathbf a_{B/I} + \mathbf a_{P/B} + \boldsymbol\omega_{B/I}\times \mathbf v_{P/B} + \boldsymbol\alpha_{B/I}\times\mathbf r_{P/B} + \boldsymbol\omega_{B/I}\times(\mathbf v_{P/B}+\boldsymbol\omega_{B/I}\times \mathbf r_{P/B})
&=\mathbf a_{B/I} + \mathbf a_{P/B} + \boldsymbol\omega_{B/I}\times \mathbf v_{P/B} + \boldsymbol\alpha_{B/I}\times\mathbf r_{P/B} + \boldsymbol\omega_{B/I}\times(\mathbf v_{P/B}+\boldsymbol\omega_{B/I}\times \mathbf r_{P/B})
\\\\
&=\mathbf a_{B/I} + \mathbf a_{P/B} + 2\;\boldsymbol\omega_{B/I}\times \mathbf v_{P/B} + \boldsymbol\alpha_{B/I}\times\mathbf r_{P/B} + \boldsymbol\omega_{B/I}\times(\boldsymbol\omega_{B/I}\times \mathbf r_{P/B})
\end{aligned} 
$$


$$\boxed{\mathbf a_{P/I} = \mathbf a_{B/I} + \mathbf a_{P/B} + 2\;\boldsymbol\omega_{B/I}\times \mathbf v_{P/B} + \boldsymbol\alpha_{B/I}\times\mathbf r_{P/B} + \boldsymbol\omega_{B/I}\times(\boldsymbol\omega_{B/I}\times \mathbf r_{P/B}) }$$

This breaks down into the components:

$$\mathbf a_{P/I}
= \mathbf a_{B/I}
+ \mathbf a_{P/B}
+ \underbrace{2\,\boldsymbol\omega_{B/I}\times \mathbf v_{P/B}}_{\text{Coriolis}}
+ \underbrace{\boldsymbol\alpha_{B/I}\times\mathbf r_{P/B}}_{\text{Euler}}
+ \underbrace{\boldsymbol\omega_{B/I}\times(\boldsymbol\omega_{B/I}\times \mathbf r_{P/B})}_{\text{Centripetal}}$$
