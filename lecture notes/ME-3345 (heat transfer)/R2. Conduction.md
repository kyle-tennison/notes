## Fourier's Law 

**Fourier's Law** expands to 2D or 3D with:

$$\mathbf q'' = k\nabla T$$

where $T$ is the scalar temperature field and $k$ is the *thermal conductivity.* This equation implicitly shows that heat transfer always moves normal to the temperature gradient; that means that an *infinitesimal* surface plotted normal to $\mathbf q''$ would lay on a larger *isotherm.* 

You can alternatively express this equation as

$$q_n'' = -k \frac{\partial T}{\partial n}$$

where $n$ is the normal to the isotherm.

## 1st Law of Thermodynamics

On a rate-basis, the 1st law of thermodynamics is:

$$\dot E_{\rm in}- \dot E_{\rm out} + \dot E_{\rm gen} = \dot E_{\rm sys}$$

I usually prefer to clump $\dot E_{\rm gen}$ with $\dot E_{\rm in}$, because energy isn't actually generated, but the book use this notation. Sometimes $\dot E_{\rm sys}$ is called $\dot E_{\rm st}$ for *energy storage* in the system.

You could equivalently express this on an interval basis, using $\Delta E$ instead of $\dot E$. 


## Heat Diffusion Equation

We often would like to solve for the temperature distribution about a solid.

### Derivation

For an isotropic solid, let us consider an individual material element with sides of length $dx, dy, dz$. 

![](excalidraw-2026-01-28-13.33.53.excalidraw.svg)
%%[🖋 Edit in Excalidraw](excalidraw-2026-01-28-13.33.53.excalidraw.md)%%

> There is also a z-dimension not shown here

In this system:

$$\dot E_{\rm in} = q_x+q_y+q_z \qquad \text{and} \qquad \dot E_{\rm out} = q_{x+dx}+q_{y+dy}+q_{z+dz}$$

We can also consider energy that might be "generated" as:

$$\dot E_{\rm gen} = \dot q \,dx\,dy\,dz$$

If the element has a approximately constant specific heat $c_p$, we can use the fact that $m c_p \Delta T = \Delta U = \Delta E_{\rm sys}$ to say:

$$\dot E_{\rm sys} = \rho c_p \frac{\partial T}{\partial t} \,dx\,dy\,dz$$
Then, putting this into the energy balance $\dot E_{\rm in}- \dot E_{\rm out} + \dot E_{\rm gen} = \dot E_{\rm sys}$ gives:

$$q_x+q_y+q_z -q_{x+dx} - q_{y+dy}-q_{z+dz} + \dot q_{\rm gen} \,dx\,dy\,dz = \rho c_p \frac{\partial T}{\partial t}\,dx\,dy\,dz$$

Because $q_{x+dx} \approx q_x + {\partial q_x}/{\partial x}\,\cdot dx$ via the Taylor series (attenuated order), we can simplify the above equation to:

$$-\frac{\partial q_x}{\partial x}dx-\frac{\partial q_y}{\partial y}dy - \frac{\partial q_z}{\partial z}dz + \dot q_{\rm sys}\,dx\,dy\,dz = \rho c_p \frac{\partial T}{\partial t}\,dx\,dy\,dz$$

Fourier's law can give us values for $q_x, q_y, q_z$ all in very similar ways. For $q_x$:

$$q_x= -k \,dy\,dz\frac{\partial T}{\partial x}$$

Which means that:

$$- \frac{\partial q_x}{\partial x} dx =  \frac{\partial}{\partial x}\left(k \frac{\partial T}{\partial x}\right) \,dx\,dy\,dz $$

Likewise:

$$- \frac{\partial q_y}{\partial y} dy =  \frac{\partial}{\partial y}\left(k \frac{\partial T}{\partial y}\right) \,dx\,dy\,dz $$

$$- \frac{\partial q_z}{\partial z} dz =  \frac{\partial}{\partial z}\left(k \frac{\partial T}{\partial z}\right) \,dx\,dy\,dz $$

Substituting this into the energy balance above, then dividing by $dx\,dy\,dz$ which appears on all terms, we get:

$$\underset{\text{\large \textbf{Heat Equation}}}{\boxed{\frac{\partial }{\partial x}\left(k \frac{\partial T}{\partial x}\right) +\frac{\partial }{\partial y}\left(k \frac{\partial T}{\partial y}\right) + \frac{\partial }{\partial z}\left(k \frac{\partial T}{\partial z}\right)+\dot q_{\rm gen} = \rho c_p \frac{\partial T}{\partial t}}}$$
^heat-eq

Each of these $\displaystyle \frac{\partial }{\partial n}\left(k \frac{\partial T}{\partial n}\right)$ terms refers to the *net heat flux input* in each $n$ axis; i.e., it is saying the same thing as $q_n-q_{n+dn}$. 


For *isotropic materials* where $k$ is constant in all directions and with temp/time, we can factor it out and instead write:

$$\frac{\partial^2T}{\partial x^2}+\frac{\partial^2T}{\partial y^2}+\frac{\partial^2T}{\partial z^2}+ \frac{\dot q_{\rm gen}}{k}= \frac{1}{\alpha} \frac{\partial T}{\partial t}$$

where $\displaystyle \alpha = \frac{k}{\rho c_p}$,  i.e. the *thermal diffusivity.* 

This equation can be abbreviated with a [scalar Laplacian](Laplace%20Operator.md#^scalar-laplacian):

$$\underset{\large \textbf{Heat Eqation} \text{ (const k)}}{\large\boxed{\nabla^2T + \frac{\dot q_{\rm gen}}{k} = \frac{1}{\alpha}\frac{\partial T}{\partial t}}}$$
^heat-eq-const-k


For cylindrical coordinates, see [here](R3.%201D%20Steady%20State%20Conduction.md#^cylindrical-heat-eq). 


### Boundary Conditions

As with any PDE, to solve this we need boundary conditions and, if solving transiently, an initial condition. 

A **Dirichlet condition** (aka, a "*boundary condition of the first kind*") fixes the temperature at a certain point. This is what is done in the [Python Implementation](#Python%20Implementation) below.

A **Neumann condition** specifies the derivative (or a multiple of it) at the boundaries. We call this a *boundary condition of the second kind* when we fix the derivative $\partial T / \partial x$ using a specific heat flux, which is simply $q''_{\rm cs} = -k \;\partial T/\partial x$. For example, a perfectly insulated wall will have $q''=0$. 

A *boundary condition of the third kind* also fixes $\partial T / \partial x$, except it is fixed with the convection equation; we call this the **convection surface condition**, which is:

$$-k \left.\frac{\partial T}{\partial x}\right|_{x=0}=h\left[T_\infty - T(0,t)\right]$$

Where the right-hand side is simply the flux form of [Newton's law of cooling](lecture%20notes/ME-3345%20(heat%20transfer)/L1.%20Introduction.md#^newtons-law-of-cooling). 




### Python Implementation

```python
from matplotlib import pyplot as plt
import numpy as np
import scipy.ndimage

# some dummy constants that seem to converge
T_top = 0
T_bot = 1
T_right = 2 
T_left = 3 
T_mid = 3
alpha = 0.005
dt = 0.01
epsilon = float('inf')
max_ep = 0.045

# create field with boundary conditions
T0 = np.zeros((100,100))
T0[:,0:3] = T_right
T0[:,-1:-3] = T_left
T0[0:3,:] = T_top
T0[-1:-3,:] = T_bot 
T0[45:55, 45:55] = T_mid

T = T0.copy()

while epsilon > max_ep:

    # continue to enforce these
    T[:,0] = T_right
    T[:,-1] = T_left
    T[0,:] = T_top
    T[-1,:] = T_bot 
    T[45:55, 45:55] = T_mid

	# solve the laplacian and timestep 
	# this implicitly assumes dt=1
    del2_T = scipy.ndimage.laplace(T)
    dT = alpha*del2_T
    T += dT

    epsilon = np.linalg.norm(dT)
    # print(f"{epsilon:.7f}  ", end="\r")

plt.imshow(T)
cbar = plt.colorbar()
cbar.ax.set_ylabel('Temperature')
plt.title("2D Heat Diffusion Equation")
plt.savefig("media/ME-3345-demo-r2_1.png")
plt.clf()
```
