## Spring Elements
### Series and Parallel Spring Elements

![[image-103.png]]

When two or more springs are *applied in parallel* (b) and have the same displacement, the equivalent spring constant is simply the sum of all springs:

$$k_e = \sum_i k_i \qquad \text{(Parallel)}$$

and Hooke's law becomes:

$$f = k_ex = (k_1+k_2+\cdots)x$$

When springs are applied *in series*, the equivalent spring constant is instead:

$$\frac{1}{k_e} = \sum _i \frac1{k_i} \quad \Rightarrow \quad k_e = \left( \sum_i k_i^{-1}\right)^{-1}\quad \text{(Series)}$$
### Static Spring Deflection

If a spring-mass system is under a uniform acceleration field (gravity), then the spring may deflect by some $\delta_{st}$ at static equilibrium; this is called the *static spring deflection*. 

If the spring is linear, the equation of motion might look something like this:

$$m \ddot x = -k(x+\delta_{st})$$
(in the simplest case). However, if we redefine $x \to x'$ such that $x'=0$ is at $x= \delta_{st}$, then we can write this as:

$$m \ddot x' = -kx'$$

In other words, we can apply Hooke's law without knowing the spring's true resting position *when the displacement of the mass is measured from the equilibrium position.* 

(You could always solve for the resting position afterwards anyways.)
