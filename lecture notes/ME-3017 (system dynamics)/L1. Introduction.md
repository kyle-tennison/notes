# L1. Introduction

A **Dynamic System** is one who's output $x_k$ depends on a previous input $u_{k-1}, u_{k-2}, \dots$. 
Usually you are finding $dx/dt$:

$$\frac{dx}{dt} = f(x_k)+u_k$$

Then, integrating this over the change in time gives us our next output:

$$x_{k+1} = \Delta T(f(x_k)+u_k)+x_k$$

For clarity, the notation is:
- $k^{\rm th}$ output: $x_k$
- $k^{\rm th}$ input: $u_k$ 

So, in a **static system**, the output $x_k$ depends *only* on $u_k$. 

## Linear and Nonlinear Systems

A **linear time-invariant** (*LTS*) system is represented by a linear ODE:

$$\cdots + \frac{d^2x}{dt^2}+a\frac{dx}{dt}+b=0$$

Notice how there are no $t$ terms here, and how the different derivatives of $x$ are not multiplied together.

If there were a time term (e.g., $a \ddot x + b\dot x + t = 0$), then the system would be *time-variant*. 

> Intuitively, I think this means that the system depends explicitly on time... and I think this means something in Landau mechanics.

If there were any mingled derivatives (e.g., $\dot x \cdot x$, $\ddot x \cdot x$, etc.), then this would make a *nonlinear* system. 

Alternatively, if the dependent variable or any of its derivatives are in a nonlinear function (e.g., $\ln \dot x$), then the system is *also nonlinear*.


### Examples

$$\frac{d^2x}{dt^2} + \big(1-\cos (2t)\big)x =0 $$

This is **time-variant** because there is an explicit time term. It is linear because the dependent variable $x$ is neither mingled with its own derivatives nor put into a nonlinear function.

$$\frac{d^2x}{dt^2}+(x^2-1)\frac{dx}{dt}+x =0 $$

This is **time-invariant** because there are no time terms, but it is nonlinear because the dependent variable is both mingled with its derivative (2nd term) and there is a nonlinear function ($x^2$) involving the dependent variable. 


### Superposition

If you have a linear function $f(t)$, *superposition* says:

$$f(x_1) + f(x_2) = f(x_1+x_2) \qquad {\rm and} \qquad f(\alpha x) = \alpha \cdot f(x)$$

---


A forced system is one with an external force applied; e.g., a mass spring damper with an external force field.

A homogeneous ODE equals zero; a non-homogeneous has a nonzero constant.

There are two types of automatic control:

- **Open-loop Control**
	- The output of the system is unknown; it has no bearing on the controller output.
	- Example: A toaster
- **Closed-loop Control** (feedback loop)
	- Like the example drawn above.
	- The system monitors the output and modifies the next input based on the behavior.
	- Example: Cruise control (PID)
